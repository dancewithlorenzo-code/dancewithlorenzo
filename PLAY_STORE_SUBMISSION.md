## 🔐 App Signing Configuration

### Step 1: Generate Keystore

```bash
# Run this command in your project root
keytool -genkeypair -v -storetype PKCS12 \
  -keystore dancewithlonrenzo.keystore \
  -alias dancewithlonrenzo \
  -keyalg RSA \
  -keysize 2048 \
  -validity 10000

# You'll be prompted for:
# - Keystore password (save securely!)
# - Key password (save securely!)
# - Your name: Lorenzo Schmidt
# - Organizational unit: Dance with Lorenzo
# - Organization: Dance with Lorenzo Ori Tahiti Japan
# - City: Tokyo
# - State: Tokyo
# - Country code: JP
```

**⚠️ CRITICAL:** Save these passwords in a secure password manager. **If you lose them, you cannot update your app!**

### Step 2: Configure Gradle (if building locally)

Create `android/gradle.properties`:
```properties
DANCEWITHLONRENZO_UPLOAD_STORE_FILE=dancewithlonrenzo.keystore
DANCEWITHLONRENZO_UPLOAD_KEY_ALIAS=dancewithlonrenzo
DANCEWITHLONRENZO_UPLOAD_STORE_PASSWORD=YOUR_KEYSTORE_PASSWORD
DANCEWITHLONRENZO_UPLOAD_KEY_PASSWORD=YOUR_KEY_PASSWORD
```

### Step 3: Build Release APK

```bash
# Using Expo
eas build --platform android --profile production

# Or locally with Expo
expo build:android -t app-bundle
```

### Step 4: Upload to Play Console

1. Go to [Google Play Console](https://play.google.com/console)
2. Create new app "Dance with Lorenzo"
3. Navigate to **Release** → **Production** → **Create new release**
4. Upload your `.aab` (Android App Bundle) file
5. Fill in release notes (see below)
